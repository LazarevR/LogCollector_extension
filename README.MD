# Log Collector

Браузерное расширение для быстрого сбора диагностической информации о текущей странице и копирования её в буфер обмена.

---

## Что собирает

| Данные | По умолчанию |
|--------|-------------|
| URL страницы | ✓ |
| Браузер и версия | ✓ |
| Размер viewport (разрешение) | ✓ |
| LocalStorage | выкл |
| SessionStorage | выкл |
| Cookie | выкл |

---

## Использование

### Сбор данных

Нажмите на иконку расширения в панели браузера — данные будут собраны и скопированы в буфер обмена автоматически.

- **Зелёный круг** снизу справа — данные успешно скопированы
- **Красный круг** снизу справа — произошла ошибка при сборе

### Настройки

Нажмите **правой кнопкой мыши** на иконку расширения → **«Настройки»**.

Откроется страница настроек, где можно включить или выключить сбор каждого типа данных. Изменения сохраняются автоматически.

## Формат скопированных данных

```
Ссылка: https://example.com/page
Браузер: Google Chrome, Версия: 124.0.6367.82 (arm64)
Разрешение: 1440x900

LocalStorage (Ключей: 2):
  token: eyJhbGci...
  theme: dark

SessionStorage (Ключей: 1):
  draft: hello world

Cookie (3):
  session_id=abc123
  _ga=GA1.2.xxx
  lang=ru
```

## Установка

### Chrome, Opera, Яндекс.Браузер

1. Откройте `chrome://extensions`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **«Загрузить распакованное расширение»**
4. Выберите папку с расширением

### Firefox

1. Откройте `about:debugging#/runtime/this-firefox`
2. Нажмите **«Загрузить временное дополнение»**
3. Выберите файл `manifest.json` из папки расширения

> **Примечание:** в Firefox временное дополнение удаляется при перезапуске браузера. Для постоянной установки расширение необходимо подписать через [addons.mozilla.org](https://addons.mozilla.org).
